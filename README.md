# RASTREADOR-PARA-MOTORISTAS
Esta versão da extensão foi otimizada para integração em aplicativos nativos Android/iOS através de WebView. A extensão mantém todas as funcionalidades originais, mas adiciona comunicação bidirecional com o aplicativo nativo.
🚀 Funcionalidades
✅ Funcionalidades Mantidas
Rastreamento GPS em tempo real
Cálculo de velocidade com alertas
Salvamento de rotas no IndexedDB
Interface responsiva otimizada para celular
Mapa Folheto interativo
🆕 Funcionalidades Adicionadas
WebView Bridge : Comunicação bidirecional com aplicativo nativo
Controle Nativo : App pode controlar funções da extensão
Notificações Nativas : Alertas enviados para o aplicativo nativo
Otimização WebView : Interface otimizada para WebView
